Introducing CodeFlux — AI-Powered Web App Builder

A full-stack platform where users can:
• Describe their app idea in natural language
• Generate complete HTML/CSS/JS instantly using AI
• Chat and iteratively refine their app
• See a live preview in real-time
• Edit and download production-ready code

⚙️ Tech Stack
• MERN (MongoDB, Express, React, Node.js)
• Google Gemini AI (for code generation)
• JWT Authentication + bcrypt security
• Vite for fast frontend development

🔥 Core Features
• Secure authentication & user management
• Project-based workflow with full CRUD
• Conversational AI interface
• Live preview using sandboxed iframe
• Code editor + downloadable output
• Version handling for generated code
